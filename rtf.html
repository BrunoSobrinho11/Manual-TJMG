Manual HTML (RTF)

Pré-requisito: Este procedimento só pode ser realizado após a conclusão do processo da Primeira Instância.

1. Backup e Preparação dos Arquivos

  1. Acesse a pasta: \\10.0.40.177\dje\RTF.
  2. Confira se existem no mínimo 298 arquivos "P".
  3. Copie **todos** os arquivos para a pasta de backup do dia, por exemplo:
    \\10.0.40.177\dje\_Backup_RTF\2023\03.Março\22.03.2023
  4. Selecione todos os arquivos "P" (exceto arquivos "PUB") e copie-os para a pasta TEMP.

2. Processamento de Belo Horizonte (024)

  1. Na pasta TEMP, recorte o arquivo de BH (024) e qualquer outro de BH de expedientes anteriores.
  2. Cole os arquivos na pasta do dia correspondente, por exemplo:
    Z:\Arquivos PUBLICACOES\Arquivos 1a instancia\2023\05.Maio\04.05.23
  3. Extraia o conteúdo dos arquivos ZIP e, em seguida, apague os arquivos ZIP.
  4. Abra e feche cada arquivo RTF para verificar se há erros. Após a verificação, coloque o mês depois do dia e compacte-os novamente.
  5. Copie o arquivo .txt da pasta do dia anterior para a pasta atual.
  6. Renomeie o arquivo .txt e o novo .zip com a data do expediente.
  7. Abra o arquivo .txt e atualize a data e o tamanho do arquivo para corresponder ao novo ZIP.

3. Transferência dos Arquivos de BH via FileZilla

  1. Conecte-se ao FILEZILLA SITE TJ_Arquivos RTF.
  2. Copie o arquivo .zip de BH para a pasta: /web1/www/arquivos/juridico/sf/diario/download
  3. Volte uma pasta para trás e copie o arquivo .txt para a pasta do dia do expediente.
  4. Acesse o site para verificar se a publicação de BH já está disponível.

4. Processamento das Demais Comarcas

  1. Volte para a pasta TEMP.
  2. Selecione todos os arquivos ZIP e extraia-os usando "Extrair aqui...".
  3. Apague todos os arquivos ZIP após a extração.
  4. Selecione todos os arquivos RTF extraídos, clique com o botão direito e use a ferramenta "Bulk Rename Here" para renomeá-los em lote.
  5. Em Add coloque no INSERT: coloque o mês do expediente. Em At Pos: deixe sempre 3.
  6. Copie todos os arquivos RTF renomeados para a pasta rtf no FileZilla.

5. Execução do Script

  1. Acesse o servidor SOLWEB-5 (via PuTTY). A senha é imprensa.
  2. Execute o primeiro comando: cd /home/publica/rotina
  3. Execute o segundo comando: ./rtf2html.pl –t
  4. No FileZilla, atualize a pasta rtf (pressionando F5) até que todos os arquivos desapareçam.
Solução de Erros: Se algum arquivo não sumir, execute o comando ./rtf2html.pl –x no SOLWEB-5 para identificar o arquivo problemático. Corrija-o na pasta TEMP (abrindo, dando Enter e Backspace, e salvando), substitua-o no FileZilla e execute novamente os comandos do passo 5.
