Jornal Administrativo

Download dos Arquivos

  1. Abra o gmail da "OPERAÇÃO" e localize a mensagem do dia anterior referente ao expediente.
  2. Clique na seta e faça o download.
  3. Salve os arquivos em uma nova pasta com a data do expediente, seguindo o caminho:
    Z:\Arquivos PUBLICACOES\Arquivos administrativos\2023\5. Maio\03.05.23
OBSERVAÇÃO: Se o e-mail "Publicação DJe" não for encontrado, contate Paula ou Michele da presidência.

Nomeando os Arquivos

  1. Navegue até a pasta do dia e abra o arquivo Word (.docx) com o LibreOffice.
  2. No LibreOffice, vá em Arquivo > Salvar como...
  3. Em "Tipo", selecione Texto em Formato Rich Text (.rtf).
  4. Nomeie o arquivo como diamestjmg01 (ex: para 25 de novembro, ficaria 2511tjmg01). Salve e feche o documento.

Enviando para o Servidor

  1. Abra o FileZilla e conecte-se ao site TJ_Arquivos RTF.
  2. Navegue até o diretório remoto: /web1/www/arquivos/juridico/sf/diario e entre na pasta rtf.
  3. Transfira o arquivo .rtf que você acabou de salvar para esta pasta RTF no servidor.

Enviando para o Site

  1. Acesse o servidor SOLWEB-5 (via PuTTY).
  2. Clique em "Bastion" e depois em "Load" e "Open" para conectar.
  3. Ao clicar em 'Open', aparecerá uma tela preta escrita "login as". Nessa hora, digite seu field, no meu caso, é field225. Aperte enter.
  4. Coloque o código ssh publica@solweb-5. Obs.: Não é permitido usar CTRL + V; copie o código e clique com o botão direito do mouse para colar. Aperte enter.
  5. Digite a senha imprensae de enter.
  6. Coloque o código cd /home/publica/rotina. Lembando que não é permitido CTRL + V; Use o botão direito do mouse para colar. Apete enter.
  7. Coloque o código ./rtf2html.pl –t. Lembando que não é permitido CTRL + V; Use o botão direito do mouse para colar.
  8. AGUARDE! Espere até que o cursor com o cifrão ($) apareça no final.

Verificação Final

  1. No FileZilla, pressione F5 para atualizar a pasta rtf. O arquivo que você enviou deve ter desaparecido.
  2. Acesse o site do DJe (Diário Judiciário Eletrônico) e verifique se a publicação foi criada corretamente.
  3. No servidor, procure pelo arquivo gerado diamestjmg01.txt e copie-o para a pasta do dia correspondente.
  4. Sua pasta vai ficar com esses Arquivos
  5. Abra o arquivo TXT, e vai para o final ctrl + End.
  6. Conte a quantidade de arquivos PDF que estão na pasta e remova os espaços dos nomes dos PDFs, se houver.
  7. Cole o código abaixo no final do arquivo TXT. Esse código será adicionado para cada PDF que estiver na pasta. No lugar dos XXX será colocado as informações dos PDFs.
    8. <p></p>

<div align="center">
    <a href="/info/pdf/index.jsp?uri=/juridico/sf/diario/download/XX.pdf"
    target="_blank"> 
        <font size="4">Acesse aqui: XXXXX </font>
    </a> 
</div>
  8. Copiar o nome dos arquivos e abri-los para copiar os títulos.
  9. Após colocar todos os PDFs no TXT, feche o arquivo e salve as alterações.
  10. Copie esse novo TXT para o servidor.
  11. Vamos enviar todos os arquivos PDFs para a pasta Download no servidor Site TJ_Arquivos RTF
  12. Abra o site do TJ e vá para o final da página para verificar se os anexos estão abrindo corretamente.
